# Hajj Checklist

A simple, beautiful checklist to help you prepare for Hajj. Track documents, clothing, prayer items, medicine, electronics, and more across organized categories.

## Features

- Three-state tracking: **Not ready** → **Available** → **Packed**
- Collapsible sections with progress indicators
- Multi-traveler support with independent checklists
- Optional Google Drive sync via Google Sheets
- Offline-first — works without internet, saves to browser storage
- Mobile-friendly responsive design
